## 使用

默认已安装以下程序：

- `git`
- `docker`
- `docker-compose` 

### 打包镜像

```bash
docker build -t yapi .
```

### 启动容器

```shell
docker-compose up -d
```

### 停止容器

```shell
docker-compose down
```
